# Photo Finish Camera GPL Source Release

This branch contains the source code and build files needed to publish the Android application source under the GPL.

## Included

- Android app source under `app/src/main`
- Product flavor resources under `app/src/demo` and `app/src/premium`
- Gradle wrapper and build configuration
- License texts bundled with the app

## Requirements

- Java 17
- Android SDK 35
- Android build tools installed through Android Studio or the command line SDK manager
